## 🎯 三个核心项目

| 项目 | 业务场景 | 核心数据 | 技术关键词 |
|------|---------|---------|-----------|
| [数据爬虫自动化平台](./projects/01-crawler-platform/) | 全国26家数据交易所产品信息采集 | **102,043条**产品数据、**26家**交易所覆盖、Playwright API拦截 | 爬虫、反爬、流式写入、OOM防护 |
| [AI金融量化信号系统](./projects/02-financial-ai/) | 基于新闻舆情的A股交易信号生成 | **+11.56%**模拟盘收益、**13层**优化→0人工审核、**双轨制**新闻分析 | DeepSeek、量化、NLTK、聚宽 |
| [AI原生社区自动化](./projects/03-community-agent/) | AI Agent在社区自主互动、学习、发帖 | **日互动20-30条**、内容A/B实验、**150条/日**限流管控 | Agent自治、内容实验、限流、去重 |

## 🛠️ 技术栈

**语言**：Python（主力）、TypeScript（OpenClaw插件）  
**AI/ML**：DeepSeek API、NLTK、scikit-learn、聚宽(JoinQuant)  
**爬虫**：Playwright、requests、随机指纹、流式写入  
**数据**：SQLite、MySQL、pandas、RAG检索  
**自动化**：OpenClaw Agent框架、cron、HEARTBEAT巡检  
**工程**：TypeScript严格模式、Result<T,E>、接口定义优先
