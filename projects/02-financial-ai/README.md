### 2. 凯利公式仓位管理

```python
# 半凯利公式，保守但可持续
def kelly_position(p=0.6, b=2.0):
    """
    p: 胜率估计
    b: 盈亏比
    f = (p*b - q) / b  →  0.5（半凯利）
    """
    q = 1 - p
    f = (p * b - q) / b  # = 0.5
    return f * 0.5  # 半凯利 = 0.25 = 25%仓位

# 实际参数
TP = 0.20   # 止盈20%

```

### 3. 聚宽全自动调仓

```python
# 关键修复：order_value不可靠 → 改用order(股数) + 3重价格兜底
def execute_order(stock, target_shares):
    current_price = get_current_price(stock)
    # 三重兜底：限价单+滑点保护+超时取消
    order(stock, target_shares, 
          price=current_price * 0.995,  # 略低于市价
          limit=True,
          timeout=30)
```
